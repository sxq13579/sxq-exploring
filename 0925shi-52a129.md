SET AUTOCOMMIT = 0：取消自动提交处理，开启事务处理

SET AUTOCOMMIT = 1：打开自动提交处理，关闭事务处理

START TRANSACTION：启动事务

BEGIN：启动事务，相当于执行START TRANSACTION

COMMIT：提交事务

ROLLBACK：回滚全部操作

SAVEBACK：设置事务保存点

ROLLBACK TO SAVEPOINT：回滚操作到保存点
